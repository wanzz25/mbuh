⚡ WANZZ INSTALLER v2.0 — REAL SSH EDITION
==========================================

REQUIREMENT:
  - Node.js versi 16 ke atas
  - Koneksi internet

CARA INSTALL & JALANKAN:
  1. Extract file ZIP ini
  2. Buka terminal di folder hasil extract
  3. Jalankan perintah berikut:

     npm install
     node server.js

  4. Buka browser dan akses:

     http://localhost:3000/wanzz-installer.html

FITUR:
  ✅ Install Pterodactyl Panel via SSH nyata
  ✅ Start Wings via SSH nyata
  ✅ Cek status VPS (Online/Offline + Ping + Uptime)
  ✅ Live log terminal real-time (SSE streaming)
  ✅ Progress bar otomatis
  ✅ Copy credentials hasil install

CATATAN:
  - VPS target harus fresh install Ubuntu 20.04 / 22.04
  - Domain panel & node harus sudah diarahkan ke IP VPS
  - Proses install membutuhkan 10-20 menit
  - Jangan tutup terminal selama proses berjalan
